Описание БД

В курсовом проекте создана БД транспортного предприятия. Набор данных хранит данные о:
- клиентах предприятия, включая данные по точкам забора и периодичность посещения;
- работниках предприятия;
- автомобильном транспорте предприятия;
- выполненных услугах и полученных платежах.
База данных позволяет:
- составлять графики работы работников;
- составлять расписание работы с клиентами на текущий день;
- получать баланс по выбранному клиенту за оказанные услуги;
- вносить записи об оказанных услугах используя расписание работы;
- проводить анализ работы используя выборки.
